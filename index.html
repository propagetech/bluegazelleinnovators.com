OUR TEAM’S CASE STUDIES

Our team of qualified and experienced professionals share their stories and goals to build our clients' success together

DATA ENGINEERING IN HEALTHCARE

OBJECTIVE: Migration of IBM-DataStage and MS-SSIS to spark

ADVANTAGES: Business users are able access to data in near real time now and the leaders are able to take data driven decisions. Business can use the data for new product ideas

WHAT WE DID: Built highly performant data ingestion pipelines from multiple sources using Apache Spark and/or Azure Databricks. Built custom ETL processes and automation in Azure Data Factory, Azure Databricks, Python, Pyspark etc. The client has now scalable and re-usable frameworks for ingesting of geospatial data sets.

DATA ENGINEERING IN PHARMACEUTICAL

OBJECTIVE: Migration of Cloudera data/informatica scripts into Spark

ADVANTAGES: Customer were able to get healthcare administration and healthcare prescriber/partner time based reports based for geo-markets, sales forecasting, customer reach, competition comparison to improve sales

WHAT WE DID: Built high performance data pipelines including data quality validation for full refresh and also incremental data load using Apache spark and Databricks. Scheduling assistance with Airflow and metadata management through aws Glue. Business user/stakeholders can use data using Amazon Athena.

DATA ENGINEERING IN DATABRICKS

OBJECTIVE: Databricks workspace cost optimizations using databricks spark logs by reducing job cost, cluster cost, standard spark configuration.

ADVANTAGES: Business users able to get report/alerts for workspace high consumption cluster, job and task. The trend analysis for workspace was beneficial for cost reduction, compute cost with the help of spark logs data by applying different methods.

WHAT WE DID: Built high performing data pipelines to handle databricks object/spark logs through event hub and captured in delta tables. Generation of report generation for variety of stakeholders using log data toward reducing the workspace
